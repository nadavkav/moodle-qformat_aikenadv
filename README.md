moodle-qformat_aikenadv
=======================

Aiken Advanced format - a simple format for creating (Any language) multiple choice questions (with  * only one correct choice, and no feedback).

Advanced features
=================
* NUMBERING - which allow the teacher to indicate one of the following numbering methods to each question: 'abc', 'ABCD', '123', 'iii', 'IIII', 'none'.
* Multi Lingual support - Allow for a local string translation to "ANSWER" & "NUMBERING" which enables smooth text input in RTL languages.
